## INTERACTION PATTERNS

### Dashboard Interactions:
```
USER ACTIONS:
├─ Click ad → see full details + recommendations
├─ Click page → drill into all angles on that page
├─ Click "Winner" badge → see scale/test recommendations
├─ Click "Brand-Aligned" tag → see why + recommendations
├─ Filter by CPA range → identify scale vs. kill threshold
├─ Sort by ROAS → identify highest efficiency
└─ Export ad IDs → use in Meta bulk operations

├─ 🟢 NEW WINNER: "AVATAR #3 hit $18 CPA — scale?" [Yes/No]

├─ 💡 RECOMMENDATION: "Untested angle on authority page" [View]
├─ 📧 EMAIL READY: "High performer → generate sequence?" [Generate]
```
